make PLATHOME=OS名　で make する．

Msys64/MinGW64 でビルドするには

make PLATHOME=mingw64 clean
make PLATHOME=mingw64 

ubuntu 22.04 でビルドするには

make PLATHOME=ubuntu22 clean
make PLATHOME=ubuntu22

macos

make PLATHOME=macos clean
make PLATHOME=macos
